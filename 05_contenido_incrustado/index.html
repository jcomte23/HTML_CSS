Contenido incrustado

El contenido incrustado se utiliza para mostrar recursos externos como, por ejemplo, mapas, previsiones meteorológicas, fórmulas matemáticas, vídeos y audios, entre otros. Este método permite utilizar un servicio o interfaz de terceros.

Etiquetas
Elemento Descripción
<img> Representa una imagen.
<iframe> Representa un contexto anidado de navegación, es decir, un documento HTML embebido.
<embed> Representa un punto de integración para una aplicación o contenido interactivo externo que por lo general no es HTML.
<object> Representa un recurso externo, que será tratado como una imagen, un sub-documento HTML o un recurso externo a ser procesado por un plugin.
<video> Representa un vídeo.
<audio> Representa un sonido.
<source> Permite a autores especificar recursos multimedia alternativos para los elementos multimedia como  <video> o <audio>.
<canvas> Representa un área de mapa de bits en el que se pueden utilizar scripts para renderizar gráficos.
<svg> Define una imagen vectorial embebida.

Etiqueta img

La etiqueta <img> se utiliza en HTML para incrustar imágenes en una página web y estas son las principales caracteristicas:

  • Las imágenes pueden insertarse en distintos formatos: .jpg, .png, .svg…
  • Cuando no se indica el ancho o alto de una imagen (ni en HTML ni en CSS) se incrusta con su tamaño por defecto.
  • Las imágenes SVG funcionan muy bien para logotipos, iconos, gráficos y diagramas. No pierden su calidad al ampliarlas o cambiarlas de tamaño. Como los archivos SVG son sólo código, tienen un tamaño reducido y se pueden comprimir aún más mediante programas específicos.

    • Los atributos más comunes en las etiquetas de imagen HTML son:

  • src: Especifica la ruta o URL de la imagen.
  • alt: Proporciona texto alternativo que se muestra si la imagen no se puede cargar.
  • title: Proporciona texto descriptivo que se muestra al pasar el cursor sobre la imagen.
  • width: Especifica el ancho de la imagen en píxeles o como un porcentaje del ancho disponible.
  • height: Especifica la altura de la imagen en píxeles o como un porcentaje de la altura disponible.
  • height: Especifica la altura de la imagen en píxeles o como un porcentaje de la altura disponible.

    • Además, puedes ver en proyectos más antiguos el uso de los siguientes atributos:

  • border: Especifica el ancho del borde alrededor de la imagen. Ya no se utiliza, mejor usar estilos CSS.
  • align: especifica la alineación de la imagen con respecto al texto circundante. Valores: left | right| middle| top| bottom. Ya no se utiliza, mejor usar estilos CSS.


    • Ejemplo:

    En el siguiente ejemplo, la imagen se carga desde el archivo imagen.jpg. Si la imagen no se puede cargar, se mostrará el texto Descripción de la imagen. Al pasar el cursor sobre la imagen, se mostrará el texto Título de la imagen. La imagen tendrá un ancho de 300 píxeles y una altura de 200 píxeles.

    <img src="imagen.jpg" alt="Descripción de la imagen" title="Título de la imagen" width="300" height="200">

    Etiqueta video

    La etiqueta <video> se utiliza en HTML para incrustar videos en una página web. Permite especificar la fuente del video mediante el atributo "src" y proporciona controles de reproducción para que los usuarios puedan reproducir, pausar, adelantar y retroceder el video.

    Atributos etiqueta Video
    Elemento Descripción
    <src> Archivo o URL del video a reproducir. Obligatoria si no tiene etiquetas dentro.
    <width> Indica el tamaño de ancho del video.
    <height> Indica el tamaño de alto del video.
    <autoplay> Comienza a reproducir el video automáticamente. OJO, está sujeto a normas del navegador.
    <loop> Vuelve a iniciar el video cuando finaliza su reproducción (en bucle).
    <muted> Establece que el video se reproduzca con sonido muteado (silenciado).
    <controls> Muestra los controles de reproducción (por defecto no se muestran).
    <playsinline> Reproduce el video en línea, es decir, en su área de reproducción.


    Ejemplo: 
          <video src="../video/HTML_bien_explicado.mp4" width="540" height="360" controls loop muted playsinline autoplay>
        Tu navegador no admite la reproducción de video.
      </video>

    Etiqueta audio

    La etiqueta <audio> se utiliza en HTML para incrustar archivos de audio en una página web. Permite especificar la fuente del archivo de audio mediante el atributo "src" y proporciona controles de reproducción para que los usuarios puedan escuchar el audio


    Ejemplo: 
          <audio src="../audio/audio.mp3" controls loading="lazy"></audio>

    Etiqueta iframe

    la etiqueta <iframe> es un elemento HTML que se utiliza para incrustar contenido de otros sitios web directamente en una página web. Este elemento es muy versátil y permite la inserción de diversos tipos de contenido, como videos, mapas y páginas web externas, dentro de la página principal. Aquí tienes una descripción más detallada de cómo funciona y para qué se utiliza.

    usos mas comunes:

  • Inserción de videos: Una de las aplicaciones más comunes es la incorporación de videos de servicios como YouTube, Vimeo o Dailymotion. Al incluir un <iframe> con el código de inserción del video, los visitantes de tu sitio web pueden ver el video directamente en tu página.
  • Integración de mapas: Puedes usar <iframe> para insertar mapas interactivos de servicios como Google Maps. Esto permite a los usuarios ver ubicaciones específicas, buscar direcciones y obtener direcciones para llegar a un lugar.
  • Inclusión de contenido externo: Puedes utilizar <iframe> para mostrar contenido de otros sitios web, como widgets de redes sociales, presentaciones de diapositivas, encuestas, formularios y mucho más. Esto permite que tu página web acceda y muestre contenido de terceros de manera integrada.
  • Visualización de contenido web embebido: Si deseas mostrar una página web completa dentro de la tuya (a menudo utilizada para incrustar contenido de blogs, noticias o sitios relacionados), puedes utilizar <iframe> para lograrlo.

    • Ejemplo: 
          <iframe src="https://www.google.com/maps/embed?pb=!1m14!1m12!1m3!1d7932.667449544543!2d-75.5835768!3d6.219648900000009!2m3!1f0!2f0!3f0!3m2!1i1024!2i768!4f13.1!5e0!3m2!1ses!2sco!4v1698232808595!5m2!1ses!2sco"
      width="600" height="450" style="border: 0" allowfullscreen="" loading="lazy" referrerpolicy="no-referrer-when-downgrade">
      </iframe>
